# FreeBSD 状态报告中文翻译格式规范

本规范用于统一全书 102 份状态报告（2001-2026）的格式、风格、术语与排版。本规范是 [CLAUDE.md](CLAUDE.md) 的补充，聚焦于分年代格式统一工作；CLAUDE.md 中已规定的事项（如 CJK 空格、路径加粗、man 页引用格式等）不再重复。

## 一、年代阶段划分

| 阶段 | 年代范围 | 文件名模式 | 上游源格式 | 主要特征 |
| --- | --- | --- | --- | --- |
| **早期** | 2001-2009 | 月份范围 `1-3.md` | SVN XML | 元信息缺"版本"项；裸 URL 多 |
| **中期** | 2010-2016 | 季度 `Q1.md` | SVN XML | "原文链接"与"原文"混用；`**链接**` 加粗 |
| **过渡期** | 2017-2020 | 混合 | SVN XML | 元信息趋于稳定；URL 后缀 `.html` |
| **近期** | 2021-2026 | 带年份前缀 `2022-Q1.md` | ADOC | 三项元信息齐全；引言无 `## 引言` 标题 |

## 二、文件命名规范

### 2.1 命名原则

- 文件名不得包含空格、中文字符或英文冒号 `:`
- 必须兼容 Windows 文件系统

### 2.2 命名模式

| 年代 | 命名模式 | 示例 |
| --- | --- | --- |
| 2001-2009 | `M-M.md` 或 `M.md`（月份范围） | `1-3.md`、`6.md` |
| 2010-2016 | `QN.md`（季度） | `Q1.md` |
| 2017 | `M-M.md`（恢复月份范围） | `1-3.md` |
| 2018 | `YYYY-MM-YYYY-MM.md`（跨年特殊命名） | `2017-10-2018-9.md` |
| 2019-2021 | `QN.md`（季度） | `Q1.md` |
| 2022 至今 | `YYYY-QN.md`（带年份前缀） | `2022-Q1.md` |

### 2.3 已知异常（待修正）

- `2024/2024-q1.md` 至 `2024/2024-q4.md`：小写 `q`，应改为大写 `Q`，统一为 `2024-Q1.md` 至 `2024-Q4.md`
  - **注意**：改名需同步更新 `
